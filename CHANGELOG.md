# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-01-XX

### Added
- Initial release
- Centralized API exception handling
- Support for all Laravel exception types
- Built-in translations (English and Arabic)
- Automatic exception handler registration
- Comprehensive configuration options
- Support for Spatie QueryBuilder exceptions
- Custom exception handlers support
- Detailed logging configuration
- Unified JSON response structure

### Features
- Auto-registration of exception handler (configurable)
- Customizable API route patterns
- Production-safe error messages
- Comprehensive exception type handling
- Translation support with easy customization
